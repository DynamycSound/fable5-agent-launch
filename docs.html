● GETTING STARTED

Your first finished task.

Fable turns a human-sized instruction into a durable workflow. Connect it to an IDE, describe the outcome, and it handles the path between.

1. Connect your workspace

Install Fable in a supported agentic IDE and authenticate with your workspace token. The integration exposes repository context only for the workflow you start.

terminal
fable login
fable connect --workspace northstar

2. Describe the outcome

Use a concrete task with clear boundaries. Fable analyzes the repository, shows its plan, then executes each step with checkpoints.

✦ A useful first prompt

“Add account export to the dashboard. Match existing permissions, add coverage, and open a PR when checks pass.”

3. Keep workflows moving

When a task needs input, a credential, or a review, its workflow pauses with the exact question. Once answered, it resumes from saved state instead of starting over.
